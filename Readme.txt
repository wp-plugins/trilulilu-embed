Plugin for showing embed movies and audio for the trilulilu.ro website. 

== Description ==

Trilulilu Embed plugin helps you post all trilulilu.ro file formats on your blog pages in a few easy steps. So for posting a trilulilu media to your blog you shoul simply add the following text to the place you want to post:
* [trilu-audio]url[/trilu-audio]
* [trilu-video]url[/trilu-video]
* [trilu-image]url[/trilu-image]
Where the url represents the webpage url showing the video or image or audio that you want to share with your readers.

== Installation ==

To install the plugin you should do the following:
1. Download and extract trilulilu-embed.zip.
1. Upload `trilulilu-embed` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
